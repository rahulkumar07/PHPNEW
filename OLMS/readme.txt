This is project is made keeping school and college lab in mind ...
Currently our project supports only C and java ...
To run this project you should have following specifications:--

1. A linux based distro   {Ours is tested on UBUNTU 10.04 64bit}
2. Latest PHP
3. Latest MySql
4. Gcc should be installed
5. Java SDK should be installed and should have a link in /usr/bin/
6. Extract the ZIP in /var/www/ or try any directory which is accessible through apache
7. open the file create_database_on_the_fly and see the instructions to create tables and database
8. open the file settings.php for your default settings. Change the settings you like.
9. For the data cleanup after program compilation please add del.sh in crontab or startup jobs which can run periodically
